# BLG483E Local Wikipedia RAG Assistant

This project is a fully local Wikipedia-based Retrieval Augmented Generation assistant.

The system ingests Wikipedia pages about famous people and famous places, chunks the documents, stores local embeddings in a vector database, retrieves relevant context, and generates grounded answers using a local language model.

## Planned Stack

- Python
- Ollama
- llama3.2:3b
- nomic-embed-text
- Chroma
- Streamlit

## Main Features

- Wikipedia ingestion
- Manual document chunking
- Local embedding generation
- Vector search with metadata filtering
- Local LLM answer generation
- Chat-style interface
- Source context display

## Project Status

Initial repository structure is prepared. Implementation will be completed sprint by sprint.
